ΥΛΟΠΟΙΗΣΗ:

Το πρόγραμμα είναι ικάνο να εκτελέσει του μηχανισμους
compression, extraction, print metadata, queries, 
print tree. Οι μηχανισμοί delete, append και συμπίεση 
με τη χρήση gzip δεν υλοποιήθηκαν λόγω πίεσης χρόνου. 
Η λειτουργικότητα του προγράμματος είναι ικανοποιήτική
διότι είναι σε θέση να εκτέλει σώστα τους μηχανισμούς
που αναφέρθηκαν. Επίσης η δόμη που υλοποίηθηκε βασίστηκε
στη λογίκη των dinodes και δουλεύει όπως πρέπει. Ακόμη 
για την ανάκτηση αρχείων/φακέλων έχουν χρησιμοποιηθεί
αναδρομικές συναρτήσεις έτσι ώστε να τηρούνται οι 
αρχικές ιεραρχίες. Τέλος τα άρχεια/φάκελοι διατηρούν
τα permissions και timestamps και μετά το extraction.
